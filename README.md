# 💉 VacinaKids — Imuni

Plataforma de acompanhamento da saúde infantil com foco na jornada de vacinação das crianças.

## 🚀 Tecnologias

- Ionic Framework
- Angular
- TypeScript
- SCSS

## 📱 Funcionalidades

- ✅ Login e autenticação
- ✅ Dashboard com painel da família
- ✅ Cadastro de crianças (CPF, RG, data de nascimento)
- ✅ Perfil individual com indicador de situação vacinal
- ✅ Histórico vacinal completo (linha do tempo)
- ✅ Carteirinha digital com QR Code
- ✅ Campanhas de vacinação ativas
- ✅ Alertas de vacinas atrasadas
- ✅ Responsivo: desktop, tablet e mobile
